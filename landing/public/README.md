# Public Assets

Place your public assets here:

- `favicon.ico` - Site favicon
- `logo.png` - Site logo (used in navbar)
- `og-image.png` - Open Graph preview image for social media sharing

These files will be copied to the root of the dist directory during build.

## Recommended Sizes

- **favicon.ico**: 32x32 or 64x64 pixels
- **logo.png**: 120x120 pixels (transparent background recommended)
- **og-image.png**: 1200x630 pixels (for social media previews)
